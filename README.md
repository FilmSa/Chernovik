## 2.1 Продуктовые метрики

Для расчетов используются показатели аудитории сервиса из п.1, а также статистические данные об использовании картографических сервисов и открытые данные о 2ГИС.

### Сводная таблица продуктовых метрик

| Метрика | Значение | Источник |
| :--- | :--- | :--- |
| **Monthly Active Users (MAU)** | 60 млн | SimilarWeb |
| **Daily Active Users (DAU)** | 20 млн | оценка (≈30% MAU) |
| **Среднее количество открытий карты** | 3 в день | аналог Google Maps |
| **Среднее количество просмотров карточек организаций** | 2 в день | сценарий поиска |
| **Среднее количество поисковых запросов** | 1.5 в день | статистика поисков |
| **Среднее количество построений маршрута** | 0.7 в день | аналог Google Maps |
| **Использование тематических слоёв** | 0.5 в день | пользовательский сценарий |
| **Загрузка оффлайн карт** | 0.02 в день | ≈1 загрузка в 50 дней |
| **Средний размер ответа карточки** | 3 КБ | оценка структуры данных |
| **Средний размер результата поиска** | 2 КБ | список организаций |
| **Средний размер маршрута** | 5 КБ | геометрия маршрута |
| **Средний размер тайла карты** | 8 КБ | векторный тайл |
